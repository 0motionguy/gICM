# /dev-refactor

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: Gemini 2.0 Flash | Also compatible with: Claude, GPT-4o


Refactors code for improved maintainability.

## Usage

```bash
/dev-refactor <file_path> [--target=readability|performance|patterns]
```

## Improvements

- Extract duplicate code
- Simplify complex functions
- Apply design patterns
- Improve naming conventions
- Reduce cyclomatic complexity
- Optimize gas usage (Web3)

## Example

```bash
/dev-refactor src/utils/tokenSwap.ts --target=performance
```
