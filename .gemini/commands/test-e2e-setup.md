# /test-e2e-setup

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: Gemini 2.0 Flash | Also compatible with: Claude, GPT-4o


Establishes end-to-end testing infrastructure.

## Usage

```bash
/test-e2e-setup [--framework=playwright|cypress]
```

## What It Does

- Installs E2E framework
- Configures test environment
- Sets up wallet mocking
- Creates sample E2E tests

## Web3 Features

- Wallet connection testing
- Transaction signing mocks
- Network switching tests
