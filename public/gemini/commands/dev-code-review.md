# /dev-code-review

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: Gemini 2.0 Flash | Also compatible with: Claude, GPT-4o


Performs comprehensive code reviews.

## Usage

```bash
/dev-code-review <file_or_directory>
```

## Reviews

- Code quality and readability
- Best practices adherence
- Security vulnerabilities
- Performance bottlenecks
- Architecture patterns
- Smart contract safety (Web3)

## Example

```bash
/dev-code-review src/components/WalletConnect.tsx
```
