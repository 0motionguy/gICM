# /performance-audit

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: GPT-4o | Also compatible with: Claude, Gemini


Analyzes application performance metrics.

## Usage

```bash
/performance-audit [--profile=lighthouse|web-vitals]
```

## Measures

- Page load times (Core Web Vitals)
- Bundle size analysis
- Memory leaks
- Render performance
- Network waterfall
- RPC call efficiency (Web3)

## Example

```bash
/performance-audit --profile=lighthouse
```
