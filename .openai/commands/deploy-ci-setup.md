# /deploy-ci-setup

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: GPT-4o | Also compatible with: Claude, Gemini


Establishes CI/CD pipelines.

## Usage

```bash
/deploy-ci-setup [--platform=github|gitlab|circle]
```

## Configures

- Build stages
- Test automation
- Deploy stages
- Environment variables
- Secrets management

## Example

```bash
/deploy-ci-setup --platform=github
```
