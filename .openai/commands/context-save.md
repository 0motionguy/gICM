# /context-save

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: GPT-4o | Also compatible with: Claude, Gemini


Saves development context for session continuity.

## Usage

```bash
/context-save [--name=session-name]
```

## Captures

- Current task state
- File changes
- Open issues
- Decision log
- Next steps
- Contract addresses (Web3)

## Example

```bash
/context-save --name=wallet-integration
```
