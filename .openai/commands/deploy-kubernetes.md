# /deploy-kubernetes

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: GPT-4o | Also compatible with: Claude, Gemini


Configures Kubernetes manifests for deployment.

## Usage

```bash
/deploy-kubernetes <app_name>
```

## Generates

- Deployment YAML
- Service YAML
- Ingress configuration
- ConfigMaps
- Secrets templates

## Example

```bash
/deploy-kubernetes blockchain-indexer
```
