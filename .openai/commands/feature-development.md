# /feature-development

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: GPT-4o | Also compatible with: Claude, Gemini


Orchestrates end-to-end feature development.

## Usage

```bash
/feature-development <feature_name>
```

## Workflow

- Creates feature branch
- Scaffolds code structure
- Implements feature logic
- Writes tests
- Updates documentation
- Creates PR
- Smart contract deployment (Web3)

## Example

```bash
/feature-development token-staking
```
