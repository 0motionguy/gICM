# /tdd-cycle

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: GPT-4o | Also compatible with: Claude, Gemini


Test-driven development workflow orchestration.

## Usage

```bash
/tdd-cycle <feature_name>
```

## Red-Green-Refactor

1. **Red**: Write failing test
2. **Green**: Write minimal code to pass
3. **Refactor**: Improve code quality

## Example

```bash
/tdd-cycle "transfer tokens"
```

Orchestrates:
- Write failing test
- Implement feature
- Refactor code
- Verify tests pass
