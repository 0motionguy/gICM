# /performance-caching

> **Universal Command**: Works across Claude, Gemini, and OpenAI.
> Optimized for: GPT-4o | Also compatible with: Claude, Gemini


Implements caching strategies.

## Usage

```bash
/performance-caching [--strategy=redis|memory|cdn]
```

## Strategies

- Redis cache setup
- In-memory caching
- CDN configuration
- HTTP cache headers
- Service worker caching
- Blockchain data caching (Web3)

## Example

```bash
/performance-caching --strategy=redis
```
